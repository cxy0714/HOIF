# Plot convergence of ATE estimates

Plot convergence of ATE estimates

## Usage

``` r
# S3 method for class 'hoif_ate'
plot(x, ...)
```

## Arguments

- x:

  Object of class hoif_ate

- ...:

  Additional arguments passed to plot

## Value

No return value, called for its side effect of drawing a convergence
plot of the ATE estimates against the order.
